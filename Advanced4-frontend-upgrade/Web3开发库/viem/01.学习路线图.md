# Viem 学习路线图

## 第一阶段：基础准备
### 1.1 前置知识
- 掌握 TypeScript 基础类型和泛型
- 了解以太坊 JSON-RPC API 规范
- 熟悉 EIP-1193 标准（以太坊 Provider 规范）

### 1.2 环境搭建
- 初始化 TypeScript 项目（推荐使用 pnpm）
- 安装核心依赖：
```bash
pnpm add viem@latest
pnpm add @types/node --save-dev
```

## 第二阶段：核心概念
### 2.1 客户端创建
- 学习配置不同链的 Client：
```typescript
import { createPublicClient, http } from 'viem'
import { mainnet } from 'viem/chains'

const client = createPublicClient({
  chain: mainnet,
  transport: http()
})
```

### 2.2 核心 API 实践
- 区块链数据读取（getBlock/getTransaction）
- 合约数据解析（decodeFunctionResult）
- 钱包操作（创建/签名交易）

## 第三阶段：进阶应用
### 3.1 智能合约交互
- ABI 类型安全集成
- 多链合约调用模式
- 事件监听与过滤

### 3.2 前端集成
- 与 React 的集成模式
- Next.js 服务端组件应用
- 响应式状态管理方案

## 第四阶段：高级主题
### 4.1 性能优化
- 批量请求处理（Batch RPC）
- 缓存策略配置
- 自定义 Transport 层

### 4.2 原理深入
- 序列化/反序列化机制
- 错误处理体系解析
- 源码结构分析

## 学习资源
1. 官方文档：https://viem.sh/
2. 实战案例库：https://github.com/wagmi-dev/viem/tree/main/examples
3. 类型定义参考：https://github.com/wagmi-dev/viem/tree/main/src/types

> **注意事项**：
> - 优先使用 TypeScript 5.3+ 版本以获得最佳类型推断
> - 开发时启用严格模式（tsconfig.json 中 `strict: true`）
> - 推荐结合 Hardhat 进行本地链测试
